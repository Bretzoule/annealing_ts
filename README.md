# annealing_ts
Simulated annealing optimization algorithm implemened in Typescript with inputs from [http://comopt.ifi.uni-heidelberg.de/software/TSPLIB95/](http://comopt.ifi.uni-heidelberg.de/software/TSPLIB95/)

## Compile and run 
You need Node (v16+) and npm (v6+) and at least one TSP Example following the parsing standards.
```
npm install
npm run
```
now the project is getting compiled 

### Add new TSP examples

Add new examples from the website : [http://comopt.ifi.uni-heidelberg.de/software/TSPLIB95/](http://comopt.ifi.uni-heidelberg.de/software/TSPLIB95/)
Create your own respecting their template or the program will crash.
